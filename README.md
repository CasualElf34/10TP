# TPs Flutter 1-10

Ce projet regroupe l'ensemble des travaux pratiques (TP) réalisés dans le cadre du cours Flutter.

## Structure du projet
- `lib/main.dart` : Configuration globale et routes.
- `lib/pages/` : Contient un fichier par TP ou par fonctionnalité majeure.
  - `home_page.dart` : Dashboard principal (TP 10).
  - `layout_page.dart` : Démonstration des layouts (TP 3).
  - `counter_page.dart` : Gestion d'état simple (TP 4).
  - `formulaire_page.dart` : Saisie et validation (TP 5).
  - `liste_page.dart` : Listes dynamiques (TP 7).
  - `api_page.dart` : Appels API REST (TP 8).
  - `firebase_page.dart` : Configuration Firebase (TP 9).

## Installation
1. Assurez-vous d'avoir le SDK Flutter installé.
2. Exécutez `flutter pub get` pour installer les dépendances.
3. Lancez l'application avec `flutter run`.
